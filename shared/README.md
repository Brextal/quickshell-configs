# quickshell-shared

Shared QML module for [Quickshell](https://quickshell.outfoxxed.me/) configurations. Provides a Pywal color palette that auto-syncs with your terminal theme.

## What's inside

- **`Pywal.qml`** — Reads `~/.cache/wal/colors.json` (generated by [pywal](https://github.com/dylanaraps/pywal)) and exposes `color0`–`color15`, `foreground`, and `background` properties. Watches the file for live changes.

## Usage

This module is symlinked into each Quickshell config (`barra/`, `wallclock/`, `calendar/`, `launcher/`) and imported as:

```qml
import "./shared" as Pywal
```

Then use `pywalColors.color4` etc. in your QML.

## Install

```bash
git clone https://github.com/Brextal/quickshell-shared.git ~/.config/quickshell/shared
```

Or run the main `install.sh` which handles this automatically.

## Dependencies

- [pywal](https://github.com/dylanaraps/pywal) or [pywal16](https://github.com/eylles/pywal16) — to generate `colors.json`
